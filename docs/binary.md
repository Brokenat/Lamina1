# Lamina 二进制字节码定义

## 全局布局

```
┌─────────────────────────────────┐
│  Magic    (4 bytes)  u32        │
├─────────────────────────────────┤
│  Version  (4 bytes)  u32        │
├─────────────────────────────────┤
│  Functions Section              │
├─────────────────────────────────┤
│  Constants Section              │
└─────────────────────────────────┘
```

## Header

| 字段    | 类型     | 大小     | 值             |
| ------- | -------- | -------- | -------------- |
| Magic   | `u32`    | 4 bytes  | `0x434D4C00`   |
| Version | `u32`    | 4 bytes  | `0x00000001`   |

## Functions Section

```
┌─────────────────────────────────┐
│  SectionSize  (8 bytes)  u64    │
├─────────────────────────────────┤
│  Func #0                        │
│  ┌───────────────────────────┐  │
│  │  BytecodeLen  (4 bytes)   │  │
│  │  Bytecode     (variable)  │  │
│  └───────────────────────────┘  │
│  Func #1                        │
│  ┌───────────────────────────┐  │
│  │  BytecodeLen  (4 bytes)   │  │
│  │  Bytecode     (variable)  │  │
│  └───────────────────────────┘  │
│  ...                            │
└─────────────────────────────────┘
```

| 字段           | 类型     | 大小         | 说明                     |
| -------------- | -------- | ------------ | ------------------------ |
| SectionSize    | `u64`    | 8 bytes      | 整个 section 的字节数    |
| BytecodeLen    | `u32`    | 4 bytes      | 当前函数的字节码长度     |
| Bytecode       | `u8[]`   | BytecodeLen  | 指令序列                 |

## Constants Section

```
┌─────────────────────────────────┐
│  SectionSize  (8 bytes)  u64    │
├─────────────────────────────────┤
│  Constant #0                   │
│  ┌───────────────────────────┐ │
│  │  Tag   (1 byte)           │ │
│  │  Data  (variable)         │ │
│  └───────────────────────────┘ │
│  Constant #1                   │
│  ┌───────────────────────────┐ │
│  │  Tag   (1 byte)           │ │
│  │  Data  (variable)         │ │
│  └───────────────────────────┘ │
│  ...                           │
└─────────────────────────────────┘
```

| 字段        | 类型     | 大小         | 说明                  |
| ----------- | -------- | ------------ | --------------------- |
| SectionSize | `u64`    | 8 bytes      | 整个 section 的字节数 |
| Tag         | `u8`     | 1 byte       | 常量类型              |

### 常量类型

| Tag | 类型   | Data 大小 | Data 布局                          |
| --- | ------ | --------- | ---------------------------------- |
| 0   | Int    | 8 bytes   | `int64_t` 小端序                    |
| 1   | Frac   | 8 bytes   | `int32_t num` + `int32_t den`，小端序 |
| 2   | Str    | 可变      | `u32 length` + `char data[length]` |
